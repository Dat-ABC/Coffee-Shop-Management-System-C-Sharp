Hướng dẫn cài đặt phần mềm Quản Lý Quán Cafe

Bước 1: Chuột phải vào tập tin Create Database.bat chọn Edit sau đó thay đổi tên server name đúng với tên server name của sql server trong máy.

Bước 2: Click đôi vào tập tin Create Database.bat để cài đặt cơ sở dữ liệu.

Bước 3: Mở thư mục Release, click đôi vào tập tin QLQuanCafe.msi để cài đặt phần mềm (Để ý chỗ lưu phần mềm cài đặt để sau khi cài đặt xong tìm vào chỗ lưu phần mềm và thay đổi chuỗi kết nối tới sql server).

Bước 4: Vào chỗ lưu phần mềm vừa cài đặt và mở file tên là: ConnectionSQL.txt thay đổi tên server name đúng với tên server name của sql server trong máy.

Sau khi cài thực hiện 4 bước trên, có thể đăng nhập vào phần mềm với tài khoản sau:
Tên tài khoản: admin
Mật khẩu: admin
